CoW Swap Apr 14, 2026 ~$1.2M
DNS hijack Registry-level takeover AI-forged docs AngelFerno

Most sophisticated AngelFerno incident to date. Attackers impersonated a senior CoW DAO contributor to Traficom (Finnish .fi TLD registry) and Gandi (AWS Route 53 .fi registrar) using forged identity docs, altered company records, fabricated AWS support transcripts, and AI-generated financial documents. Domain transferred to NETIM, pointed at attacker Cloudflare nameservers, wildcard Let's Encrypt cert issued. Users hitting cow.fi got a phishing replica that harvested wallet connections and seed phrases via fake unlock modals.

Blockaid → dApp Scanning. Catches the drainer SDK regardless of how the user arrived at the page. Resilient to any registrar-side cleverness.

SE talk track: "Your brand is the target. Registrar security alone lost CoW $1.2M of user funds. AI-forged paperwork is now part of the playbook."

eth.limo Apr 17, 2026 $0 (DNSSEC blocked)
DNS hijack EasyDNS compromise ENS gateway AngelFerno

Attackers socially engineered eth.limo's EasyDNS account, pointed NS first through Cloudflare, then to Namecheap-hosted attacker infrastructure. DNSSEC saved the day — validating resolvers returned SERVFAIL instead of attacker records. Strategic significance: eth.limo is an ENS gateway; a successful hijack would have weaponized every ENS-hosted frontend behind it. AngelFerno is escalating from per-project targets to shared crypto infrastructure.

Blockaid → dApp Scanning. Catches the drainer when DNSSEC isn't present (which is <1% of DeFi projects).

SE talk track: "DNSSEC is the single control that saved eth.limo. Less than 1% of DeFi projects publish it. dApp scanning is the layer that catches it when they don't."

Steakhouse Financial Mar 30, 2026 Pre-empted
DNS hijack Registrar social engineering AngelFerno

Between 05:00–10:00 UTC, every DNS record for steakhouse.financial was replaced. Nameservers moved from OVH (stable since 2022) to Hostinger's dns-parking.com, A record moved to shared hosting in Germany, MX records moved off Google Workspace, new Let's Encrypt cert issued at 08:57 UTC. Attacker had full control over web and email.

Blockaid → Threat Intel Feed + dApp Scanning. Proactive detection — Blockaid notified Steakhouse and partner wallets before user losses materialized.

OpenEden + Curvance Feb 16, 2026 Contained
DNS hijack GoDaddy social engineering AngelFerno

Attackers socially engineered a GoDaddy employee for unauthorized access to both project accounts. Domains transferred to Namecheap, NS pointed at malicious IPs on web-hosting.com (162.254.39.77, 199.188.201.151), fresh SSL certs issued same-day for valid HTTPS. Both frontends served AngelFerno drainer JavaScript.

Blockaid → dApp Scanning. Flagged the malicious JS before drainer SDK could execute. Blockaid-protected wallets across all integrations stayed safe through the incident window.

Bybit Feb 2025 $1.4B
Supply chain UI tampering Lazarus / DPRK

Lazarus compromised Safe{Wallet}'s frontend infrastructure via a phished developer account, then pushed JS that activated only when Bybit's cold-wallet address connected. Signers on the legitimate Safe domain saw a routine transfer; what they signed was a delegatecall upgrade that handed multisig control to the attacker. Largest crypto theft on record.

Blockaid → Cosigner. Independent out-of-band path catches the divergence between intent and signed payload — the supply-chain attacker can't reach the second validation.

Ledger Connect Kit Dec 2023 ~$600K (100+ dApps)
Supply chain NPM compromise Angel Drainer

Angel Drainer poisoned the @ledgerhq/connect-kit NPM package (versions 1.1.5–1.1.7) via a phished former Ledger employee. ~100+ dApps including hey.xyz, sushi.com, zapper.xyz auto-pulled the malicious version and served the drainer.

Blockaid → dApp Scanning. Detected autonomously in ~6 minutes. MetaMask, OpenSea, Zerion, Rainbow users protected from the second the payload landed. First team to identify and disclose the full scope.

Proof point: "When Ledger Connect Kit was poisoned, Blockaid detected it in 6 minutes — without human intervention. Our customers were safe before anyone else even knew."

Drainer ecosystem
Industrial-scale phishing and drainer-as-a-service
Inferno / AngelFerno (cumulative) Ongoing $250M+
Drainer-as-a-service Phishing dApps Permit / Approval farming Address spoofing

Industrial-scale drainer rented to affiliates as a SaaS product. Operated across 16,000+ phishing sites at peak; fee-collector wallet keeps ~15%, affiliates ~85%. Six categories of malicious tx span: native transfers (wrong-address), ERC-20 transfer farming, approval farming (often unlimited), permit / EIP-712 farming, stealthy verified contracts (e.g., claim functions crediting arbitrary recipients), and address spoofing via vanity prefixes.

Blockaid → Endpoint Protection (Tx + dApp Scanning). Drainer SDK fingerprints catalogued in real time across all integrations; ~10% of dApps observed are malicious — flagged before signature. Simulation handles permits and stealthy contracts that calldata parsing misses.

Blockaid → dApp Scanning. Drainer JS detectable on legit domains; simulation also catches the malicious approval intent.

Drift Protocol Apr 1, 2026 ~$285M
Squads multisig signer compromise DPRK / Lazarus

Solana perpetuals DEX. Treasury secured by a Squads multisig (Solana equivalent of Gnosis Safe). Attackers compromised enough signer devices (malware via fake job offers / poisoned dev tools — same cluster as Radiant Oct 2024) to hit threshold. With valid signatures in hand, a single transaction drained three vaults in under one minute. Funds reached Binance within hours. Chainalysis flagged the tx in ~9 hours. Second-largest Solana hack ever.

Blockaid → Cosigner. Sits between the multisig and the chain. Decodes actual transaction bytes (not what the wallet UI claims) and refuses pre-signature when destination is new, value is abnormal, or pattern unprecedented. Detection in ~0.3 seconds.

Internal pitch: "Drift = why you need a cosigner on privileged key operations. Radiant Capital was the same group, same pattern, same fix."
SE talk track: "Anytime a prospect mentions a multisig, Fireblocks, Copper, BitGo, Squads, or any self-custodied treasury — open with Drift. 'Your contracts are fine. The question is what happens when an attacker has the signatures.'"

KelpDAO rsETH Bridge Apr 18, 2026 $292M + $177M Aave bad debt
1-of-1 DVN RPC compromise + DDoS failover Forged cross-chain packet

LayerZero bridge configured with a single verifier (1-of-1 DVN). Attackers compromised both upstream RPC nodes the verifier read from, then DDoSed the primary to force failover onto the poisoned backup. Verifier signed a forged lzReceive packet claiming 116,500 rsETH had been locked on Ethereum — nothing was. Counterfeit rsETH minted on 20+ chains and posted as collateral on Aave / Compound / Euler to borrow another $236M. Largest DeFi hack of 2026.

Blockaid → Monitoring & Risk Exposure. Bridge-supply reconciliation (wrapped asset on destination vs. locked collateral on origin) flags 116,500-rsETH divergence within seconds. Pause-trigger on lending markets blocks the downstream $236M bad-debt cascade.

Internal pitch: "KelpDAO = why every protocol that depends on a bridged asset needs independent bridge monitoring, and why 1-of-1 DVNs are negligence."
SE talk track: "Use with any LayerZero / Wormhole / CCTP / cross-chain protocol. Also effective with lending markets — Aave's $8.45B TVL outflow in 48 hours is the contagion argument."

Moonwell on Base Feb 15, 2026 $918K
Oracle misconfiguration Wrong Chainlink feed

Compound V2 fork on Base. Chainlink publishes two cbETH feeds: a USD price (~$2,216) and an ETH ratio (~1.12). Moonwell's ChainlinkOracle was written for USD only — no handling for ETH-denominated ratios. An admin pointed cbETH at the ratio (1.12) instead of USD (2,216). Oracle reported cbETH at $1.12 — 1,974× undervalued. Attacker flash-loaned $565 of WETH, deposited as collateral, borrowed $925,614 of cbETH (oracle-valued at $468), swapped on Uniswap at real prices, repaid the flash loan, kept $918K. Drain in <1 minute.

Blockaid → Monitoring & Risk Exposure. Price sanity bands fire on oracle values outside expected range. "cbETH oracle returning $1.12" flagged within seconds — long before 67 days of pre-positioning matters. Pre-tx validation API also classified the exploit transaction as exploit_contract_execution / severe.

Internal pitch: "Moonwell is the cleanest case for lending protocols. Not a novel attack. Not a smart-contract bug. An admin picked the wrong feed from a dropdown — exactly what a sanity check catches."
SE talk track: "Use with any lending market — Aave, Compound, Morpho, Silo, Euler, Spark. Pair with Rhea Finance and Blend (Stellar) for pattern reinforcement."

Dinari usdPlus Jan 8 & 27, 2026 Backdoor on 3 proxies
Uninitialized proxy Spoofed-upgrade impl Mempool front-run

Tokenized-stocks platform on Base. Dinari deployed proxy contracts without immediately calling initialize(). Attacker bot spotted the unclaimed proxy in mempool, front-ran with their own initialize() to become owner, and immediately upgraded the proxy to a malicious implementation. The sophisticated twist: the malicious impl was designed to spoof subsequent upgrade events — if Dinari tried to "fix" the proxy, event logs would show a clean upgrade but storage would remain unchanged, leaving the backdoor live while appearing remediated. Three silent state-modification txs followed 3 weeks later.

Blockaid → Cosigner + Monitoring. Cosigner inspects initialize() and upgradeTo(), refusing pre-signature when caller has no prior relationship or impl is unverified / freshly deployed. Proxy-upgrade monitoring tracks EIP-1967 implementation slots and fires on storage changes that don't match emitted events — catches the spoofed-upgrade trick.

Internal pitch: "Dinari is the textbook 2026 proxy hack. Same shape as Parity 2017, with a spoofed-upgrade layer that fools any team relying on Etherscan events for verification."
SE talk track: "Use with any protocol using upgradeable contracts (most of them). Stablecoin / RWA issuers: 'Your token proxy is a $N-million honeypot.'"
